مَلَابِسُ المَدْرَسَةِ Les vêtements d’école

Clare Verbeek, Thembani Dladla, Zanele Buthelezi

Mlungisi Dlamini, Ingrid Schechter

Heba Tesheh, Maaouia Haj Mabrouk

Mashael Muhanna

Arabic

Level 1

هَذَا الفُسْتَانُ طَوِيلٌ.

Cette robe est longue.

هَذِهِ الكَنْزَةُ كَبيرَةٌ.

Ce pull est grand.

هَذِهِ الحَقيبَةُ كَبيرَةٌ.

Ce sac est grand.

هَذَا الحِزامُ…

Cette ceinture est...

هَذِهِ القُبَّعَةُ صَغِيرَةٌ.

Ce chapeau est petit.

هَذِهِ الجَوَارِبُ قَصِيرَةٌ.

Les chaussettes sont courtes.

لَكِنَّ هَذَا الحِذَاءَ جَديدٌ.

Mais ces souliers sont neufs.

إِنَّهُ مُنَاسِبٌ جِدًّا…

... ils font bien.

Written by: Clare Verbeek, Thembani Dladla, Zanele Buthelezi
Illustrated by: Mlungisi Dlamini, Ingrid Schechter
Translated by: Heba Tesheh, Maaouia Haj Mabrouk
Read by: Mashael Muhanna
Language: Arabic
Level 1
